# 地牢探险 (Dungeon Quest)

俯视角实时动作 Roguelike 地牢探险游戏，基于 Python 3.13 + Pygame 2.6 开发。

## 游戏简介

勇闯随机生成的 5 层地下城，击败每层的 Boss，收集武器与护甲，最终击败恶魔领主逃出生天！

## 操作说明

| 按键 | 功能 |
|------|------|
| W A S D | 移动角色 |
| 鼠标移动 | 瞄准方向 |
| 鼠标左键 | 攻击（近战 / 远程） |
| E | 拾取物品 / 打开背包 |
| F | 进入下一层（击败Boss后在楼梯上） |
| ESC | 暂停 / 继续 |

## 游戏系统

### 武器 (20种)
- **近战武器 15把**：木棒、短剑、铁剑、手斧、长矛、战斧、弯刀、骑士剑、流星锤、火焰剑、冰霜刃、暗影匕首、巨剑、圣剑、龙牙、雷锤
- **远程武器 5把**：短弓、猎弓、连弩、魔法弹珠、毁灭者
- **4个稀有度**：普通(绿) → 稀有(蓝) → 史诗(橙) → 传说(红)
- 属性包括：攻击加成、范围/宽度、暴击率、暴击伤害、吸血

### 护甲 (4种)
- 皮甲(30%减伤) → 锁子甲(50%) → 铁板甲(65%) → 龙鳞甲(80%)

### 敌人
- **史莱姆**：慢速近战，血量较高
- **骷髅射手**：远程弹幕，保持距离
- **暗影刺客**：极快近战，血量低但攻速快

### Boss（每层一个）
- **F1 巨型史莱姆王**：慢速近战，半血分裂成3只史莱姆
- **F2 骷髅将军**：近战 + 周期性召唤骷髅射手
- **F3 暗影领主**：极快速度 + 瞬移背后3连击
- **F4 炎魔**：远程环形弹幕 + 保持距离风筝
- **F5 恶魔领主**：冲刺 + 扇形弹幕，半血狂暴

### 难度递增
- 怪物血量/攻击：×1.3^(楼层−1)
- 怪物数量：F1=30 → F2=45 → F3=66 → F4=99 → F5=150
- 高楼层掉落更高稀有度装备

## 运行方式

### 方式一：直接运行（推荐）
双击 `dist/DungeonQuest.exe` 即可运行，无需安装任何环境。

### 方式二：源码运行
```bash
pip install pygame
python main.py
```

## 技术实现

- **地图生成**：BSP（二叉空间分割）算法随机生成房间 + L 形走廊
- **视野系统**：Bresenham 射线检测实现战争迷雾 + 记忆地图
- **战斗系统**：扇形近战判定 + 弹幕系统，支持暴击、吸血、击退
- **AI 状态机**：IDLE → PATROL → ALERT → CHASE → ATTACK
- **打包**：PyInstaller 单文件 exe，内置中文字体
